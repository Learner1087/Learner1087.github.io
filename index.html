Final Year Project

WiFi Fingerprint-Based Indoor Localization System — Ongoing

Overview

An end-to-end indoor positioning system that uses WiFi signal strengths (RSSI) from surrounding access points to predict a user's (x, y) coordinates inside a building. The system uses a 1D Convolutional Neural Network trained on collected WiFi fingerprints.

Python PyTorch FastAPI Kotlin Jetpack Compose ESP32 / C++ Machine Learning

Backend — FastAPI Server

  • Training pipeline: loads WiFi scan CSV data, builds RSSI vectors, normalizes to [0,1], augments with Gaussian noise, trains a 1D-CNN (Conv1D → BatchNorm → ReLU → MaxPool → FC → Dropout) with early stopping
  • REST API endpoints: /positions (save scans), /predict (CNN inference), /path (A* pathfinding), /facilities (POI lookup), /maps (floor plans)

Android App — Kotlin + Jetpack Compose

  • 5 screens: WiFi List, Location/Scan (data collection), Floor Plan, Predict (real-time localization), Show Stored
  • Real-time localization with Kalman filter smoothing for noisy predictions
  • Filters out locally-administered (randomized) MAC addresses for fingerprint stability
  • Networking via Ktor HTTP client; local storage via SharedPreferences

Embedded Firmware — M5StickC Plus (ESP32)

  • M5_wifiscan: Handheld device for training data collection — scans 2.4 GHz + 5 GHz, records ~120 scans per location, uploads via HTTP POST
  • M5_wfi_predictor: Real-time position prediction device with single-shot and continuous scanning modes

Key Technical Choices

  • WiFi fingerprinting — leverages existing infrastructure, no extra hardware needed
  • 1D-CNN — captures correlations between RSSI values across APs; outperforms fully-connected networks
  • Active scanning — faster AP discovery (~1–2s vs ~3s passive)
  • Kalman filter — smooths noisy predictions on the Android client
  • Data augmentation — Gaussian noise during training improves real-world generalization

HKBU Career Fair Companion App

iOS Prototype — SwiftUI + MVVM · Built for HKBU Career Fair 2026 (Mar 10–12)

Overview

A full-featured iOS companion app for the HKBU Career Fair, supporting 100+ exhibitor booths. Includes authentication (Auth0 SSOID), personalized recommendations, interactive floor maps, QR-based e-stamp gamification, and push notifications.

Swift SwiftUI MVVM Auth0 AVFoundation PDFKit

Core Features

  • Smart Recommendations: Scoring algorithm (70% industry/zone match, 30% purpose/preference) recommends booths based on user profile
  • E-Stamp Gamification: QR code scanning at booths to collect stamps across 3 categories (Recruitment, Branding, Skill-Up) with 4 prize tiers
  • Interactive Map: Visual floor plans for 3 venues across 3 days with server-driven booth layouts
  • Exhibitor Directory: Search and filter 100+ booths by date, zone, interest, and purpose
  • Events & Itinerary: Event listing with chronological sorting, bookmarking with push notification reminders
  • QR Code System: Pipe-delimited strings with signature verification for booth stamps, event stamps, and prize redemption
